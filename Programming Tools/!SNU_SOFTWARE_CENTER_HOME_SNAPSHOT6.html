SNU software development center

This page is the hub for the development of SNU software, and the download of non-game, non-malware software. For the malware section, refer to the SNU malware center, for the gams section, refer to the SNU Games center.

This page is intended for avid and professional computer users and software developers, however anyone can join if they can contribute, or if they are looking for a particular piece of software. Discussion is also allowed.

Notice!

This page is being reconstructed, as it is a very old page with high importance. It was created before 3.0.4, and snapshot 1 used the outdated SNU starter syntax. Snapshots weren't even a thing on the site yet when this page was being developed. Almost everything has changed since then.

Renovations are going to be made so that this page gets to the level of importance that it deserves. It has been abandoned for far too long.
